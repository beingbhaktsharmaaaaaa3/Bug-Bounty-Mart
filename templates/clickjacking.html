This page is intentionally missing X-Frame-Options header

CTF Challenge

Why is this vulnerable?

Without X-Frame-Options: DENY or Content-Security-Policy: frame-ancestors 'none',
this page can be embedded in an iframe on any attacker-controlled website.

An attacker can overlay invisible elements to trick users into clicking
buttons or links they didn't intend to click.

How to exploit:

  1. Create an HTML page with an iframe pointing to this URL
  2. Position the iframe and overlay a transparent div on top
  3. Trick the user into clicking what appears to be a harmless button
  4. The click actually triggers the action on this framed page
