BugBountyMart API

Our REST API allows developers to integrate with our platform.

Authentication

Include JWT token in Authorization header:

Authorization: Bearer <token>

Obtain token via POST /api/login

Endpoints

MethodEndpointDescription
POST/api/loginAuthenticate user
POST/api/registerRegister new user
GET/api/user/<id>Get user profile
GET/api/search?q=<query>Search products
POST/graphqlGraphQL endpoint

Example: Login

curl -X POST http://127.0.0.1:5000/api/login   -H "Content-Type: application/json"   -d '{"username":"alice","password":"password"}'
